==========================================
  B. ADVANCED FEATURES (PLANNED / NEW)
==========================================

14. [PLANNED] Live Crowd Density Heatmap Inside Temple Premises
    - Visual floor plan of the temple with color-coded zones:
      sanctum sanctorum, queue corridors, prasad counters, parking areas
    - Updated every 15 minutes from CCTV people count + prediction model
    - Zone-level occupancy: Green (< 40%), Yellow (40-70%), Red (> 70%)
    - Integration: extend /api/v1/cctv with camera_id -> zone mapping
    - Frontend: interactive SVG floor plan on Dashboard page
    - Backend: new model ZoneMapping (camera_id, zone_name, coordinates)

15. [PLANNED] Festival Surge Predictor with Countdown
    - Calendar view showing upcoming festivals with predicted footfall surge
    - Surge percentage vs normal day, e.g. "Aadi Pooram: +340% expected"
    - Countdown timer to next high-risk festival day
    - Historical comparison: this year vs last year for same festival
    - Integration: extend /api/v1/events with surge_pct field
    - Use existing FootfallHistory + prediction model with festival flags
    - Frontend: new FestivalSurge component on Dashboard + Events page

16. [PLANNED] Queue Wait Time Estimator per Entry Gate
    - Given predicted footfall + number of open gates, estimate wait time per gate
    - Display: "North Gate: 23 min wait | East Gate: 8 min wait | Redirect here"
    - Factors: current footfall, gate throughput rate, time of day
    - Integration: new model EntryGate (name, throughput_per_min, is_open)
    - New endpoint: GET /api/v1/crowd/queue-times
    - Frontend: queue time cards on Dashboard + dedicated Queue page

17. [PLANNED] Staff Deployment Recommender
    - Based on predicted footfall for next 6 hours, recommend exact staffing
    - "Deploy 4 queue marshals at North Gate from 9-11 AM, reduce to 2 post noon"
    - Role types: queue marshal, prasad counter, parking attendant, security
    - Exportable as daily shift schedule PDF
    - Integration: new model StaffRole + StaffSchedule
    - New endpoint: GET /api/v1/staff/recommendations?date=YYYY-MM-DD
    - Frontend: new StaffPlanner page with printable schedule

18. [PLANNED] Anomaly Alert System - "Something's Off Today"
    - When actual footfall deviates > 25% from prediction, flag it
    - Manual gate count input or sensor-driven comparison
    - Auto-suggest reason: "Footfall 40% above forecast. Possible unannounced
      religious event or VIP visit. Consider opening emergency queues."
    - Push notification to admin users
    - Integration: compare FootfallHistory.footfall vs prediction output
    - New endpoint: GET /api/v1/alerts (active anomalies)
    - Frontend: alert banner on Dashboard + notification bell in Nav

19. [PLANNED] Devotee Flow Optimization
    - Suggest optimal visit times based on prediction: "Visit between 2-4 PM
      for shortest wait (est. 12 min vs 45 min at peak)"
    - Route suggestion through temple to minimize congestion
    - Integration: combine queue estimator + crowd heatmap
    - Frontend: visitor-facing widget on Landing + Bookings page

20. [PLANNED] Revenue Analytics Dashboard
    - Combined view: booking revenue + donation income over time
    - Festival vs non-festival revenue comparison
    - Top seva slots by revenue, top donation causes
    - Monthly/quarterly trend charts
    - Integration: aggregate from existing Booking + Donation models
    - Frontend: new Revenue page with charts (Chart.js or Recharts)

21. [PLANNED] SMS / WhatsApp Notifications
    - Booking confirmation with QR code
    - Queue alert: "Your turn in ~15 minutes"
    - Festival reminder: "Thai Pongal is tomorrow, expected crowd: High"
    - Anomaly alert to temple admins
    - Integration: Twilio / WhatsApp Business API

22. [PLANNED] Smart Parking with ANPR
    - Automatic Number Plate Recognition via CCTV cameras
    - Auto vehicle entry/exit without manual input
    - Pre-booking parking slots for festival days
    - Integration: extend existing parking system with ANPR model
    - New endpoint: POST /api/v1/parking/anpr-detect

23. [PLANNED] Multilingual Voice Assistant
    - Tamil + English voice commands for devotees
